# getAllRentData

This function will return all rent NFT data.

Response (array)

* nftAddress (address)
* tokenId (uint256)
* rentFee (uint256)
* feeTokenAddress (address)
* rentFeeByToken (uint256)
* isRentByToken (bool)
* rentDuration (uint256)
* renterAddress (address)
* renteeAddress (address)
* serviceAddress (address)
* rentStartTimestamp (uint256)
